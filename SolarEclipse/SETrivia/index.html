1. Saros Cycles

1.1 Saros Cycle Arithmetic 1

Before you read this, you need to know about Saros Cycles. When a Saros Cycle Number is subtracted by 58, eg 139 - 58 = 81, you will get the same pattern but the paths are different and the gamma is shifted upwards a bit, eg member 2 is 0.002 -> 0.0015 (these are just imagination).


1.2 Saros Cycle Arithmetic 2

When you subtract or add by 3, eg 139 - 3 = 136, the staring or ending point for the cycle is switched, 139 northen -> southern, 136 southern -> northern. The gamma is flipped and shifted upwards when added by 3, flipped and downward when subtracted by 3.
